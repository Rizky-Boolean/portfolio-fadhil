PROJECT

Grade Sales

TECHNOLOGY

HTML, CSS, Javascript, MySQL, Bootstrap, PHP, NodeJS, ExpressJS

YEAR

2024

DESCRIPTION

Schedule Grade Sales adalah aplikasi backend yang dirancang untuk menghitung grade atau peringkat sales berdasarkan total penjualan selama 3 bulan sejak sales tersebut mulai bergabung. Proses perhitungan dilakukan secara otomatis setiap tanggal 1 setiap bulannya untuk periode 3 bulan sebelumnya, sehingga evaluasi kinerja sales menjadi lebih konsisten dan terjadwal tanpa intervensi manual.

Aplikasi ini dibangun menggunakan Node.js dengan framework Express serta didukung oleh Node Cron untuk penjadwalan otomatis dan MySQL sebagai basis data. Aplikasi berjalan di lingkungan Windows Server dan dirancang untuk stabilitas dan performa jangka panjang dalam proses analisis dan klasifikasi data penjualan.
